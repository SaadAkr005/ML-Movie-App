# ML Movie App

## Overview

Allows users to explore movies, and recieve personalised recommendations. 

The application focuses on delivering a fast, responsive, and intuitive user experience while leveraging modern web technologies.

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite

## Planned Features

* Movie search
* Movie details pages
* Responsive design
* Movie recommendation system
* API integration

## Installation

```bash
git clone https://github.com/SaadAkr005/ML-Movie-App
cd ml-movie-app
npm install
npm run dev
```
